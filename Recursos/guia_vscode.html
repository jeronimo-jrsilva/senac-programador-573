0. Preparação

Crie os 3 arquivos base. Todas as extensões usam esses mesmos arquivos para teste.

Passo 1: Crie uma pasta

Crie uma pasta chamada teste-extensoes no seu desktop. Abra ela no VS Code (Arquivo → Abrir Pasta).

Passo 2: Crie o arquivo index.html

Clique com o botão direito na pasta → Novo Arquivo → index.html. Cole o código abaixo:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Meu Site</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p id="intro">Este é meu primeiro site.</p>

    <div class="cartao">
        <h2>Card de Teste</h2>
        <p>Conteúdo do card.</p>
    </div>

    <script src="app.js"></script>
</body>
</html>

Passo 3: Crie o arquivo style.css

body {
    font-family: sans-serif;
    background: #f8fafc;
    padding: 20px;
}

.cartao {
    border: 1px solid #ccc;
    padding: 20px;
    border-radius: 8px;
    margin-top: 16px;
}

Passo 4: Crie o arquivo app.js

const nome = "Aluno";
const nota = 8;

if (nota >= 7) {
    console.log(nome + " aprovado");
} else {
    console.log(nome + " reprovado");
}

1. Live Server

Ritwick Dey — Abre seu HTML no navegador com atualização ao vivo.

📦 Instalar

  1. Pressione Ctrl+Shift+X (abre o painel de extensões)
  2. Busque por Live Server
  3. Clique em Instalar (o autor é Ritwick Dey)

🧪 Testar

  1. Clique com o botão direito no index.html
  2. Selecione "Open with Live Server"
  3. O navegador abre automaticamente — veja seu site!
  4. Mude o texto do <h1> no VS Code → pressione Ctrl+S
  5. Observe: o navegador atualizou sozinho, sem F5!

2. Auto Rename Tag

Jun Han — Renomeia tags HTML de abertura e fechamento juntas.

📦 Instalar

  1. Ctrl+Shift+X → busque Auto Rename Tag
  2. Clique em Instalar (autor: Jun Han)

🧪 Testar

  1. No index.html, coloque o cursor em <h1>
  2. Mude h1 para h2
  3. Observe: o </h1> mudou automaticamente para </h2>!
  4. Experimente com <p>, <div>, <span>

3. Prettier

Prettier — Formata seu código automaticamente ao salvar.

📦 Instalar

  1. Ctrl+Shift+X → busque Prettier
  2. Clique em Instalar (autor: Prettier)
  3. Depois, ative "Format on Save": Ctrl+, → busque "format on save" → marque a opção

🧪 Testar

  1. Abra o style.css
  2. Embaralhe a indentação manualmente (coloque tudo na mesma linha, por exemplo)
  3. Pressione Ctrl+S
  4. Observe: o código se alinhou sozinho!

4. Error Lens

Alexander — Mostra erros e warnings direto na linha do código.

📦 Instalar

  1. Ctrl+Shift+X → busque Error Lens
  2. Clique em Instalar (autor: Alexander)

🧪 Testar

  1. Abra o app.js
  2. Mude a linha const nome = "Aluno"; para const nome = ; (apague o valor)
  3. Observe: aparece uma mensagem vermelha na mesma linha do erro
  4. Agora experimente: const x = 10 (sem ponto e vírgula) — veja o warning amarelo
  5. Desfaça as mudanças com Ctrl+Z

5. HTML CSS Support

ECEC — Autocompleta classes CSS dentro do HTML.

📦 Instalar

  1. Ctrl+Shift+X → busque HTML CSS Support
  2. Clique em Instalar (autor: ECEC)

🧪 Testar

  1. No index.html, dentro do <div class="">, coloque o cursor entre as aspas
  2. Pressione Ctrl+Espaço
  3. Observe: as classes definidas no style.css (cartao) aparecem na lista!
  4. Selecione cartao — a classe foi inserida sem erro de digitação
