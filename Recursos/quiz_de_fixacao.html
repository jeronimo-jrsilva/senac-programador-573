Seção 1 — Algoritmos

O que é um algoritmo? Como o computador pensa?

📌 Conceito Rápido

Um algoritmo é uma sequência de passos ordenados e finitos para resolver um problema. Computadores seguem algoritmos à risca — sem improvisar, sem pular etapas.

Seção 2 — Variáveis e Tipos de Dados

Caixinhas que guardam informação na memória

📌 Conceito Rápido

  • let → cria uma variável que pode mudar de valor
  • const → cria uma constante que nunca muda
  • string → texto, sempre entre aspas: "João"
  • number → número sem aspas: 25, 3.14
  • boolean → só dois valores: true ou false

Leia o código e preveja o que acontece

Seção 5 — Repetição (Loops)

while e for — automatizando a repetição

📌 Conceito Rápido

  • while → repete enquanto a condição for true. Você controla quando para.

Seção 6 — Arrays e Coleções

Listas ordenadas e dicionários chave-valor

📌 Conceito Rápido — Arrays

  • O índice sempre começa em 0 (zero), nunca em 1
  • .push() → adiciona no fim
  • .pop() → remove o último
  • .unshift() → adiciona no início
  • .shift() → remove o primeiro
  • .includes(x) → retorna true se x existir na lista
  • .splice(i, q, novo) → remove q itens a partir do índice i e insere novo
