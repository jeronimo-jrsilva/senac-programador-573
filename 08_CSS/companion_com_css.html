2. Parágrafos e Formatação de Texto

Este é um parágrafo comum. O navegador já aplica um espaçamento entre parágrafos automaticamente.

Tags de formatação inline (dentro do texto):

Negrito — tag <strong>. Também pode ser <b>, mas prefira strong.

Itálico / ênfase — tag <em>. Alternativa: <i>.

Texto destacado — tag <mark> (como um marca-texto amarelo).

Sublinhado — tag <u>.

Texto pequeno — tag <small> (notas legais, avisos).

Riscado — tag <del> (texto removido/excluído).

Texto com sobrescrito (sup) e subscrito (sub) — útil para fórmulas: H2O ou x2.

Código inline — tag <code> (fonte monoespaçada).

9. Comentários em HTML

Comentários são anotações no código que o navegador ignora — não aparecem na página.

Veja o código fonte desta página (Ctrl+U) para encontrar os comentários.

Sintaxe HTML: <!-- texto do comentário -->

Comentários servem para:

  • Documentar seções do código
  • Desabilitar temporariamente partes do HTML
  • Deixar lembretes para você e sua equipe

9b. Comentários em CSS

No CSS, os comentários usam /* texto */. Eles também são invisíveis na tela.

Exemplo (veja no código fonte do <style>):

/* ============================================
   SECÃO: CABEÇALHO
   ============================================ */
h1 {
    color: blue;  /* cor do título principal */
}

Assim como no HTML, servem para organizar e documentar o código CSS.
